# UnitConverterApp

A simple **Windows Unit Converter Application** built to convert values between different measurement units.

This application supports conversions for:
- **Length** (meters, centimeters, inches, etc.)
- **Weight / Mass**
- **Temperature**
- **Volume**
- **Time**

It displays conversion results instantly without requiring internet connectivity and runs locally on your system with low resource usage.
